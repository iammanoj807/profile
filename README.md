# Manoj Kumar Thapa Portfolio

This is the modern React + Vite version of the portfolio.

## Project Structure

- `src/components/`: Reusable React components (Hero, Skills, etc.)
- `src/data/`: Static data (portfolio content)
- `src/hooks/`: Custom React hooks
- `src/utils/`: Utility functions
- `src/App.jsx`: Main application layout
- `src/main.jsx`: Entry point

## Commands

- `npm install`: Install dependencies
- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run preview`: Preview production build

## Migration Notes

The original single-file `app.js` has been refactored into modular components.
Global styles are in `src/index.css`.
